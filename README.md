# Workshop

Denna lilla workshop erbjuder en mängd olika utmaningar att lösas i grupp. De går ifrån 0-3, där 0 är lite mer av uppvärmningsövningar och 1-3 lite mer avancerade.

## Starta

```bash
npm i
```

För att köra varje utmaning så kör följande:

```bash
npm run c:0
npm run c:1
npm run c:2
npm run c:3
```

Läs instruktionerna för varje utmaning i respektive mapp.
